## ✅ Sobre o projeto

Projeto que foi desenvolvido para o desafio do projeto ONE que consiste na parceria entre a Alura e a Oracle. O projeto é da trilha de Java aonde consiste em um conversor de moedas que converte o valor de uma moeda para outra.
